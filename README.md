## 🚀 "Gold Standard" Feature: Confirmed IDOR (BOLA)

You can now perform **Multi-User IDOR tests** to eliminate false positives.

### How to use:
1. Obtain two auth tokens: one for **Victim** account and one for **Attacker** account.
2. Run the scanner with the `--auth-attacker` flag:

```bash
python src/apiscanner/cli.py \
  --target https://api.yourtarget.com \
  --auth "Bearer VICTIM_TOKEN" \
  --auth-attacker "Bearer ATTACKER_TOKEN" \
  --output report.json
```

**How it works:**
The scanner finds resources belonging to the Victim (Token 1) and then attempts to access them using the Attacker's identity (Token 2). If successful, it reports a **CRITICAL** vulnerability with 100% confirmation.

## ☁️ Deploying to Cloudflare

Your new **Enterprise Dashboard** can be hosted on **Cloudflare Pages** in seconds.

### Quick Deploy (Drag & Drop):
1. Log in to your [Cloudflare Dashboard](https://dash.cloudflare.com/).
2. Navigate to **Workers & Pages** > **Create application** > **Pages**.
3. Select **Upload assets**.
4. Drag the `dashboard/` folder into Cloudflare.
5. Once deployed, you will get a professional URL (e.g., `api-scanner-pro.pages.dev`).

### Usage:
Simply open your Cloudflare URL and upload the `report.json` generated by the CLI. You will see a beautiful, interactive analytical view of your scan results.
